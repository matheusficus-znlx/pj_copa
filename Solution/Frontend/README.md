# Frontend

## Ambiente

Crie um arquivo `.env` a partir de `.env.example` se quiser sobrescrever a URL da API.

Valor padrão recomendado:

```env
VITE_API_BASE_URL=/api
```

Com isso, o Vite usa proxy local para o backend em `http://localhost:5000`.

## Execução

```bash
npm install
npm run dev
```
